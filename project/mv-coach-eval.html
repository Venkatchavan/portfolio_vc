MV-Coach-Eval — HAR Benchmarking Platform

Production-Grade Human Activity Recognition Evaluation Harness

View on GitHub

Project Overview

A Multimodal Virtual Coach Evaluation Harness — a production-grade benchmarking platform for Human Activity Recognition (HAR) systems. Features comprehensive evaluation including accuracy/F1 metrics, Expected Calibration Error (ECE), robustness testing with noise and missing data, Leave-One-Subject-Out (LOSO) generalization analysis, and Monte Carlo Dropout uncertainty quantification. Built with a config-driven architecture using Hydra, a pluggable model registry, Docker support, and targeting ≥80% test coverage.

Technologies Used

  • Python 3.10+ & PyTorch
  • Hydra Configuration Framework
  • Monte Carlo Dropout
  • Scikit-learn & NumPy
  • Docker Containerization
  • Makefile Build Automation

Key Features

  • Comprehensive metrics: accuracy, F1, precision, recall, ECE
  • Robustness testing: noise injection, missing data simulation
  • LOSO cross-validation for subject-independent analysis
  • Monte Carlo Dropout uncertainty quantification
  • Pluggable model registry with config-driven pipelines

Technical Challenges

  • Designing universal evaluation for diverse HAR architectures
  • Monte Carlo Dropout uncertainty at scale
  • Reproducible experiments via Hydra configuration
  • Robust evaluation under noisy/incomplete sensor data

Evaluation Capabilities

  • Expected Calibration Error for prediction confidence
  • Adversarial perturbation testing
  • Leave-One-Subject-Out generalization metrics
  • Automated benchmarking with Docker & reproducibility
  • ≥80% test coverage target

Results & Impact

The platform provides a rigorous, reproducible evaluation framework for HAR systems that goes beyond standard accuracy metrics. By incorporating calibration, robustness, generalization, and uncertainty analysis, it gives researchers and practitioners a holistic understanding of model performance and reliability in real-world deployment scenarios.
