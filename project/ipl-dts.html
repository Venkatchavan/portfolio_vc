T20 Decision Intelligence Platform (IPL)

Analyst-grade sports analytics platform on IPL 2008–2025 ball-by-ball data with custom metrics, calibrated ML models, and offline RL strategy engine

View on GitHub Live Demo

Project Overview

This is not a fan dashboard — it is a decision intelligence platform built to answer analyst-grade questions about IPL T20 cricket. Using 2008–2025 ball-by-ball data, it computes 12 custom per-delivery metrics, trains 4 calibrated XGBoost models, and includes an offline RL Strategy Lab powered by Fitted Q-Iteration (FQI).

Representative questions it answers: Which batting actions in which match states were historically associated with better outcomes? How does a player's strike rate change under critical pressure vs neutral pressure? Was Team X's bowling allocation in death overs optimal given their WPA patterns?

Technologies Used

  • Python + Pandas + NumPy (data pipeline)
  • XGBoost with Platt + Isotonic calibration
  • Fitted Q-Iteration (FQI) — offline batch RL
  • Scikit-learn (feature engineering, cross-validation)
  • Streamlit (10-tab interactive dashboard)
  • Pytest (pipeline unit tests)

12 Custom Metrics

  • WPA — Win Probability Added (per delivery)
  • ESA — Expected Score Added (per delivery)
  • PI — Pressure Index (4-component score 0–100)
  • SDS — State Difficulty Score
  • CER — Contextual Economy Rate (bowler, venue-adjusted)
  • DSI — Death Suppression Index (bowler composite)
  • MLS — Matchup Leverage Score (batter vs bowler style)
  • BCR — Boundary Conversion Rate + PCR, VSI, PRR, Control Rate

ML Models

  • EFS — Expected Final Score (XGBoost Regressor)
  • CWP — Chase Win Probability (XGBoost + Platt calibration)
  • BWP — Ball-level Wicket Probability (XGBoost + Isotonic)
  • PCR — Phase Collapse Risk (XGBoost + Platt)
  • Time-aware split: Train 2008–2022, Val 2023, Test 2024–2025

Offline RL Strategy Lab

  • Algorithm: Fitted Q-Iteration (FQI) — offline batch RL
  • Actions inferred from historical ball-level behaviour
  • Recommendations reflect historically associated outcomes, not causal optima
  • Transparent framing: low-confidence states (<20 obs) flagged
  • 10-tab Streamlit dashboard: Match State Engine, Team DNA, Player Value, Pressure Profiles, Matchup Intelligence, Decision Audit, Scouting & Role Fit, Strategy Lab

Results & Impact

The platform demonstrates how advanced sports analytics can surface genuinely novel insights beyond standard batting averages and economy rates. The per-delivery WPA and ESA metrics reveal which individual balls and decisions most changed match outcomes. The RL Strategy Lab shows what historically optimal batting looks like in specific game states — with honest framing about its limitations as a recommendation system. Deployed as a live Streamlit app with full methodology documentation inside the dashboard itself.
