SocialMedia AD Agency — AI Ad Creation SaaS

Production-Grade Multi-Agent SaaS for Automated Affiliate Ad Creation

View on GitHub

Project Overview

A production-grade SaaS platform that automates the entire affiliate ad creation workflow using CrewAI multi-agent orchestration. Seven specialized AI agents handle market research, content strategy, copywriting, visual design, compliance review, A/B testing, and platform publishing. Built with FastAPI REST API, PostgreSQL with Alembic migrations, Redis caching, multi-tenancy, Stripe billing, and support for multiple LLM providers (OpenAI, Gemini, Anthropic, Mistral). Achieves 74% test coverage across 430 tests.

Technologies Used

  • Python 3.12+ & CrewAI
  • FastAPI REST API
  • PostgreSQL & Alembic Migrations
  • Redis Caching
  • OpenAI, Gemini, Anthropic, Mistral
  • Docker & Stripe Billing

Key Features

  • 7 CrewAI agents (Market Researcher, Content Strategist, Copywriter, Visual Designer, Compliance Reviewer, A/B Tester, Publisher)
  • Multi-LLM support: GPT-4, Gemini, Claude, Mistral
  • Full SaaS: multi-tenancy, Stripe billing, JWT auth, RBAC
  • Automated compliance across Meta, Google, TikTok
  • 430 tests at 74% coverage

Technical Challenges

  • Orchestrating 7 autonomous agents with inter-agent dependencies
  • Ad compliance across multiple platform policies
  • Multi-tenant SaaS with secure data isolation
  • LLM provider integration with fallback strategies

Architecture Highlights

  • CrewAI agent orchestration pipeline
  • FastAPI with async endpoints
  • PostgreSQL with Alembic migration management
  • Redis for caching & rate limiting
  • Docker containerized deployment

Results & Impact

The platform successfully automates the complete ad creation lifecycle from market research to multi-platform publishing. The multi-agent architecture enables parallel processing of different ad creation phases, while the compliance engine ensures regulatory adherence across major advertising platforms. With 430 tests and 74% coverage, the system demonstrates production-grade reliability and maintainability.
