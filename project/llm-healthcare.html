Onco-Variant-Guard — Neuro-Symbolic Cancer AI

Neuro-Symbolic Cancer Variant Classification with Hallucination-Free LLM Explanations

View on GitHub

Project Overview

An end-to-end neuro-symbolic AI system for cancer variant classification using the Kaggle Personalized Medicine dataset. Combines TF-IDF neural classification (59.3% accuracy) with 300+ automatically extracted symbolic rules, a knowledge graph (3,101 nodes, 25,908 edges), and a hallucination-free LLM system powered by Llama 3.2 3B. Features a 5-layer hallucination prevention system (RAG retrieval from 454K chunks, knowledge graph validation, rule-based validation, evidence-only prompts, response validation), PII redaction, prompt injection detection, and neuro-symbolic consensus predictions.

⚠️ Research prototype for educational purposes only. Not intended for clinical use or medical diagnosis.

Technologies Used

  • Python 3.10+ & Scikit-learn
  • FAISS (454K vector index)
  • Sentence-BERT Embeddings
  • Llama 3.2 3B via Ollama
  • NetworkX Knowledge Graphs
  • PyTorch & RAG Pipeline

Key Features

  • TF-IDF baseline at 59.3% accuracy with 300+ auto-extracted rules
  • Knowledge graph: 3,101 nodes, 25,908 edges (Genes→Variants→Drugs→Cancers)
  • 5-layer hallucination prevention with evidence-only LLM generation
  • RAG retrieval: 454,918 chunks with 0.80+ similarity scores
  • Safety: PII redaction, prompt injection detection, 26 comprehensive tests

Technical Challenges

  • Zero-hallucination LLM constrained to dataset-verified facts only
  • Extracting 300+ interpretable rules from complex medical data
  • Building knowledge graph from unstructured clinical literature
  • Balancing neural accuracy with symbolic interpretability

System Architecture

  • RAG retrieval → Knowledge Graph query → Rule retrieval
  • Grounded LLM generation (evidence-only prompts)
  • 5-layer validation & confidence assessment
  • Neuro-symbolic consensus: Neural + Symbolic + Evidence
  • 245 gene rules + 31 keyword rules + 19 tree rules

Results & Impact

The system achieves 59.3% validation accuracy on 9-class cancer variant classification (matching Kaggle baselines) while providing fully interpretable explanations through neuro-symbolic integration. The hallucination-free LLM system achieves <5% hallucination rate with 100% citation coverage. The knowledge graph successfully maps 2,656 variants, 245 genes, 163 drugs, and 11 cancer types with their relationships, enabling evidence-grounded clinical queries.
