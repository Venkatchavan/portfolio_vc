Project Overview

A Python-based desktop application for conducting giveaways from YouTube live chat. It provides real-time monitoring, smart filtering, random winner selection, and exportable audit trails.

Key Features

  • Live mode: monitor YouTube chat with YouTube Data API v3
  • Offline mode: import chat logs from CSV/text files
  • Smart filtering: by keywords, blacklist, message count
  • Random winner selection with weighted options
  • Data export: winners and participant logs in CSV

Technologies Used

Python Tkinter YouTube Data API v3 CSV Export Tools

Technical Challenges

  • API management: Managing real-time YouTube API rate limits while maintaining responsive chat monitoring
  • Fair selection: Implementing fair and transparent winner selection algorithms with proper randomization
  • Performance optimization: Handling large chat volumes during popular streams without performance degradation
  • User interface: Creating user-friendly desktop interface for streamers with intuitive controls and clear feedback
